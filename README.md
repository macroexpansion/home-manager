# Nix home-manager

Use `home-manager`:
```bash
home-manager switch
```

add path to `fish`:
```bash
fish_add_path /nix/var/nix/profiles/default/bin
fish_add_path /opt/homebrew/bin
fish_add_path $HOME/go/bin
fish_add_path $HOME/nvim-macos/bin
fish_add_path $HOME/.cargo/bin
fish_add_path $HOME/.nix-profile/bin
```
